Conditional Sentences

Mempelajari kalimat bersyarat untuk mengungkapkan kemungkinan dan harapan

Apa itu Conditional Sentences?

Conditional sentences adalah kalimat yang mengandung kondisi dan hasil. Digunakan untuk mengungkapkan kemungkinan, harapan, dan situasi yang tidak nyata.

0

Type 0

Fakta umum dan kebenaran universal

If: Present Simple

Result: Present Simple

1

Type 1

Kemungkinan nyata di masa depan

If: Present Simple

Result: Future Simple

2

Type 2

Situasi tidak nyata di masa kini

If: Past Simple

Result: Would + V1

If: Past Perfect

Result: Would have + V3

Conditional Type 0

Struktur:

If + Present Simple, Present Simple

Contoh:

If you heat water to 100°C, it boils.

Ciri-ciri:

  • • Kebenaran universal - selalu benar
  • • Fakta ilmiah - hukum alam
  • • Kebiasaan - rutinitas yang konsisten
  • • Tidak ada masa depan - berlaku untuk semua waktu

Contoh Kalimat

Fakta Ilmiah:

If you freeze water, it becomes ice.

(Jika Anda membekukan air, itu menjadi es)

Kebiasaan:

If I drink coffee at night, I can't sleep.

(Jika saya minum kopi di malam hari, saya tidak bisa tidur)

Kebenaran Umum:

If you don't study, you don't learn.

(Jika Anda tidak belajar, Anda tidak belajar)

Rutinitas:

If it rains, the streets get wet.

(Jika hujan, jalanan menjadi basah)

Conditional Type 1

Struktur:

If + Present Simple, will + V1

Contoh:

If it rains tomorrow, I will stay at home.

Ciri-ciri:

  • • Kemungkinan nyata - bisa terjadi
  • • Kondisi masa depan - belum terjadi
  • • Hasil yang mungkin - realistis
  • • Harapan atau rencana - masa depan

Contoh Kalimat

Rencana:

If I have time, I will visit you.

(Jika saya punya waktu, saya akan mengunjungi Anda)

Prediksi:

If you study hard, you will pass the exam.

(Jika Anda belajar keras, Anda akan lulus ujian)

Perintah:

If you see John, tell him to call me.

(Jika Anda melihat John, minta dia menelepon saya)

Saran:

If you feel tired, you should rest.

(Jika Anda merasa lelah, Anda harus istirahat)

Conditional Type 2

Struktur:

If + Past Simple, would + V1

Contoh:

If I were rich, I would buy a mansion.

Ciri-ciri:

  • • Situasi tidak nyata - imajiner
  • • Harapan atau impian - tidak realistis
  • • Kondisi yang berbeda - berbeda dari kenyataan
  • • Menggunakan "were" - untuk semua pronoun

Contoh Kalimat

Impian:

If I had wings, I would fly to the moon.

(Jika saya punya sayap, saya akan terbang ke bulan)

Harapan:

If I were you, I would take that job.

(Jika saya menjadi Anda, saya akan mengambil pekerjaan itu)

Skenario Alternatif:

If we had more time, we would visit the museum.

(Jika kita punya lebih banyak waktu, kita akan mengunjungi museum)

Saran Imajiner:

If I knew the answer, I would tell you.

(Jika saya tahu jawabannya, saya akan memberitahu Anda)

Conditional Type 3

Struktur:

If + Past Perfect, would have + V3

Contoh:

If I had studied harder, I would have passed the exam.

Ciri-ciri:

  • • Penyesalan - tentang masa lalu
  • • Skenario yang berbeda - tidak sesuai kenyataan
  • • Kritik terhadap masa lalu - harusnya berbeda
  • • Menggunakan past perfect - untuk kondisi

Contoh Kalimat

Penyesalan:

If I had known about the party, I would have come.

(Jika saya tahu tentang pesta itu, saya akan datang)

Kritik:

If you had been more careful, you wouldn't have made that mistake.

(Jika Anda lebih berhati-hati, Anda tidak akan membuat kesalahan itu)

Skenario Alternatif:

If we had left earlier, we would have caught the train.

(Jika kita berangkat lebih awal, kita akan menangkap kereta)

Harapan yang Hilang:

If I had had more money, I would have bought that car.

(Jika saya punya lebih banyak uang, saya akan membeli mobil itu)

Ringkasan Conditional Sentences

Type Kondisi Hasil Makna Contoh
Type 0 Present Simple Present Simple Fakta umum If you heat ice, it melts.
Type 1 Present Simple will + V1 Kemungkinan nyata If it rains, I will stay home.
Type 2 Past Simple would + V1 Situasi tidak nyata If I were rich, I would travel.
Type 3 Past Perfect would have + V3 Penyesalan masa lalu If I had known, I would have helped.
